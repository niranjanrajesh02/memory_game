• SASTA GUITAR HERO

Game Instructions

  • The aim of the game is to press the matching key when the ball makes contact with the sink
  • Play the game in an isolated environment without external distractions
  • The complete session will last around 40 minutes
  • There are 7 levels in the game with a 20 second break between each level
  • Click the “Finish Session” button only when you have finished all 7 levels and are prompted to do so
  • Your performance in the game will be recorded to help in our research
  • We expect the game to be completed in one sitting
  • Press Spacebar to pause and resume the game
